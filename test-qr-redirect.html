QR Code Redirect Test

This page tests the QR code redirect functionality for the EdUTEND system.

How it works:

  1. Lecturers generate QR codes with redirect information
  2. Students scan the QR code with their camera
  3. The system automatically redirects to the attendance page
  4. Attendance is marked automatically

Test QR Code with Redirect

This QR code contains redirect information that should automatically take students to the attendance page.

Expected Behavior: When scanned, this QR code should:
  • Show a success notification
  • Redirect to Student.html#attendance
  • Automatically mark attendance

Test QR Code without Redirect (Legacy)

This QR code uses the old format without redirect information.

Expected Behavior: When scanned, this QR code should:
  • Show the scan results
  • Require manual "Mark Attendance" button click

Test Instructions

  1. Open the Student.html page in another tab
  2. Navigate to the QR Scanner section
  3. Scan the QR codes above using your device's camera
  4. Observe the different behaviors
